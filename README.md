1. 프로젝트 목적: C++의 기본적인 기능을 이용해서 간단한 Text RPG 게임을 제작


2. 공부 기간: 2022/12/00 (약 1일)


3. 사용한 기술: C++, 비주얼스튜디오


4. 주요 기능:


    체력, 공격력, 방어력을 가진 3가지 캐릭터와 몬스터 구현


    사용자 입력에 따른 캐릭터 선택 및 번갈아서 공격 구현
